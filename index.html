Flexbox

O flexbox é um modelo de layout unidimensional que permite organizar e alinhar os elementos dentro de um elemento pai, chamado de container. O flexbox é composto por um conjunto de propriedades que podem ser aplicadas ao container e aos seus elementos filhos, chamados de itens. O flexbox facilita a criação de layouts responsivos e adaptáveis para diferentes tamanhos de tela.

 #Style.css         
    .conteiner{
        height:200px ;
        width: 700px;
        background-color: rgb(187, 187, 187);
        border: 1px solid #000000;
        display: flex;
        flex-direction: column;
    }

A propriedade define em que direção o contêiner quer empilhar os itens flex.flex-direction

O valor empilha os itens flexíveis verticalmente (de cima para baixo):column
