Flexbox

Documentação

Display: flex

A primeira propriedade que precisa ser declarada para que o Flexbox funcione!

Algumas propriedades como flex-direction, justify-content e align-content já vem com os valores padrão

Display Flex
flex-direction valor
justify-content valor
flex-start valor

Exemplo

1
2
3
 #Style.css
            
    .conteiner{
            height:200px ;
            width: 700px;
            background-color: rgb(187, 187, 187);
            border: 1px solid #000000;
            display: flex;
    }

Propriedade Valor
flex-direction valor
justify-content valor
flex-start valor
